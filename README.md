# Student Grade Management System

## Overview

This Python project implements a simple student grade management system that allows users to add students, update their grades, and calculate average performance. The system stores data in a SQLite database for persistence and supports operations such as course enrollment, grade recording, GPA calculation, and reporting.

---

## Section 2: Project Objectives

### Objectives

1. Create a user-friendly interface to manage student records.
2. Calculate individual and class-wise average grades.
3. Generate performance reports.
4. Store and retrieve student data persistently.

---

## Section 3: Features Implemented

### Features

* Add new student records with name and email
* Add new course records with instructor and credits
* Enroll students in courses
* Record and update student grades
* Calculate GPA for each student
* Display student transcripts
* Generate class-wise statistics and reports
* Store data persistently using SQLite database

---

## Section 4: Technologies Used

### Technologies Used

* Python 3.8+
* SQLite3 (database)
* OOP concepts (classes and objects)
* SQL queries (SELECT, INSERT, UPDATE, JOIN)
* Datetime module (for timestamps)

#### Prerequisites

* Python 3.8 or higher installed on your system


## Section 6: How to Run the Project

### How to Run

1. Open terminal or command prompt.
2. Navigate to the project folder.
3. Run the main script:

```
python main.py
```

4. Follow the on-screen menu to add students, enroll in courses, record grades, or view reports.
5. To exit, select the exit option from the menu.
